# Student Task Manager

A simple web-based Student Task Manager built using HTML, CSS, and JavaScript.

This project was created to practice Git and GitHub concepts including:

- README.md
- Branching
- Merge Conflicts
- Pull Requests (PR)
- Forking
- Issues
- .gitignore
- Open Source Workflow

---

## Features

✅ Add tasks

✅ Delete tasks

✅ Mark tasks as completed

✅ Simple user interface
